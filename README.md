# Academic Planner

A full-stack academic productivity platform built using React, Express.js, PostgreSQL, and Prisma.

## Features

* Subject Management
* Task Tracking
* Deadline Calendar
* Pomodoro Study Timer
* Study Analytics Dashboard
* PostgreSQL Database Integration

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Vite

### Backend

* Express.js
* Node.js
* Prisma ORM

### Database

* PostgreSQL

## Setup

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
node src/server.js
```
